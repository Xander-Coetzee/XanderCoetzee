# Xander Coetzee - Personal Portfolio

Welcome to my personal portfolio website! This project showcases my skills, experience, and education as an IT student at North-West University. It serves as a central hub for my professional identity, featuring a downloadable CV and contact information.

## About

This website is a digital resume and portfolio designed to demonstrate my proficiency in web development and my passion for technology. It provides visitors with an interactive and engaging way to learn more about my background, technical skills, and professional journey.

## Features

- **Interactive UI**: Engaging user interface with hover effects on skill items and buttons.
- **Smooth Transitions**: Seamless fade-in and fade-out animations when navigating between pages.
- **Responsive Design**: Compatible with various screen sizes and devices.
- **Downloadable CV**: Easy access to my full Curriculum Vitae in PDF format.
- **Contact Integration**: Direct links to my LinkedIn and GitHub profiles, and a copy-to-clipboard feature for my email address.
- **Dynamic Content**: JavaScript-powered interactions for a modern user experience.

## Technologies Used

This project utilizes the following technologies:

- **HTML5**: For the structural foundation of the website.
- **CSS3**: For styling, including Flexbox for layout and custom animations.
- **JavaScript**: For DOM manipulation, event handling, and page transitions.
- **Font Awesome**: For high-quality icons.
- **Google Fonts**: For modern and readable typography (Roboto).

## Project Structure

- `index.html`: The main landing page.
- `resume.html`: Detailed resume section covering education, experience, and skills.
- `contact.html`: Contact page with email and social media links.
- `style.css`: Global styles and animations.
- `resume.css`: Specific styles for the resume page.
- `contact.css`: Specific styles for the contact page.
- `CV-Xander Coetzee.pdf`: Downloadable CV file.

## Setup & Usage

To view the website locally:

1.  Clone the repository or download the source code.
2.  Navigate to the project directory.
3.  Open `index.html` in your preferred web browser.
